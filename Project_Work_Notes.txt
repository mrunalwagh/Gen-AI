I have used this commands for genral operation like- creating requirements file, fastAPI load, etc.

python -m venv <virtual env>      # used for creating virtual env
<virtual env>\Scripts\activate    # for activate venv
pip freeze > requirements. txt    # used for all requirements
uvicorn main:app                  # for running the fast api
uvicorn main:app --reload         # for countine running whenver you save file
pip install -r requirements       # for install all requirements
